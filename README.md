# Interview Prep

full-stack project built with React (Vite) and Node.js (Express).

## Prerequisites

- **Node.js**: v25.6.1 or higher
- **npm**: v11.9.0 or higher

## Local Development

The project is split into two directories: `frontend` and `backend`. You will need to open two terminal windows to run both servers simultaneously.

### 1. Running the Backend

```bash
npm install

npm run dev
```

### 2. Running the Frontend

```bash
npm install

npm run dev
```
